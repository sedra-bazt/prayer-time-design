# 🕋 Islamic Prayer Times Application

A dynamic, real-time web application built with pure web technologies that fetches and displays accurate daily prayer times. The application integrates an external Islamic API to provide prayer schedules based on the user's location and calculation methods.

---

##  Tech Stack

- **Core Logic:** Vanilla JavaScript (ES6+ Fetch API, Async/Await)
- **Structure:** HTML5 (Semantic elements)
- **Styling:** Modern CSS3 
- **Data Source:** Aladhan Prayer Times API
